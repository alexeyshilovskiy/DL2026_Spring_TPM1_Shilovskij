# Глоссарий терминов Book24.ru

Термины, используемые в анализе:

| Термин | Определение | Где используется |
|--------|-------------|------------------|
| **Авторизация** | Процесс входа пользователя в существующий аккаунт | Раздел 1.2 (оформление заказа) |
| **Бестселлеры** | Книги, которые продаются лучше всего в данный период | Раздел 1.1 (описание главной страницы) |
| **Избранное / Отложенные** | Раздел, где пользователь может сохранять книги для будущей покупки | Раздел 1.2 (шаг 7) |
| **Карточка товара** | Страница с подробной информацией о конкретной книге | Раздел 1.2 (шаг 6), раздел 1.3 |
| **Корзина** | Раздел, где собраны товары, готовые к оформлению заказа | Раздел 1.2 (шаги 7-9) |
| **Корректура текста** | Исправление опечаток и ошибок в поисковых запросах | Раздел 1.3 (проблема поиска) |
| **Конверсия** | Переход пользователя к целевому действию (например, покупке) | Раздел 2.2 (стратегическая цель) |
| **Нишевая литература** | Книги для узкой аудитории, не являющиеся массовыми бестселлерами | Раздел 1.3 |
| **Новинки** | Книги, недавно поступившие в продажу | Раздел 1.1 (описание главной) |
| **Ознакомительный фрагмент** | Несколько страниц книги, доступные для просмотра до покупки | Раздел 1.3 |
| **Опечатка** | Ошибка в написании слова при поиске | Раздел 1.3 |
| **Отзывы** | Мнения пользователей о прочитанных книгах | Раздел 1.2, раздел 1.3 |
| **Поисковый запрос** | Текст, который пользователь вводит в строку поиска | Раздел 1.2, раздел 1.3 |
| **Промокод** | Специальный код для получения скидки или бонуса | Раздел 1.2 (корзина) |
| **Пустой поисковый запрос** | Ситуация, когда поиск не находит ни одного товара | Раздел 1.3 |
| **Регистрация** | Создание нового аккаунта на сайте | Раздел 1.2 |
| **Хиты** | Самые популярные книги (синоним бестселлеров) | Раздел 1.1 |
| **Эффект листания** | Возможность пролистать несколько страниц книги онлайн | Раздел 1.3 |
