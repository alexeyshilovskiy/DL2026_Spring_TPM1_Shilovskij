# Анализ пользователей Book24.ru

## Общая информация
Book24.ru — интернет-магазин издательской группы Эксмо-АСТ. 
**Издательства:** «Эксмо», «АСТ», «Манн, Иванов и Фербер», «Азбука», «Альпина», «Синдбад».
**Дополнительные бренды:** «БОМБОРА», «Corpus», «Mainstream», «Редакция Елены Шубиной», «Вилли-Винки», «Fanzon», «Комильфо», «Канц-Эксмо».

## Сегменты пользователей

### 1. Активные читатели (ядро аудитории)
| Характеристика | Описание |
|----------------|----------|
| **Демография** | 25-45 лет, преимущественно женщины (65-70%) |
| **Поведение** | Покупают 3+ книги в месяц, следят за новинками, участвуют в программе лояльности |
| **Сценарии использования** | • Еженедельные визиты на сайт<br>• Использование «Отложенных» и «Избранного»<br>• Чтение отзывов и рейтингов |
| **Потребности** | Быть в курсе новинок любимых авторов, получать персонализированные рекомендации |
| **Боли** | Однотипные рекомендации, сложно отслеживать новинки |

### 2. Покупатели подарков и нишевых товаров
| Характеристика | Описание |
|----------------|----------|
| **Демография** | 20-55 лет, гендерный баланс 50/50 |
| **Поведение** | Покупают 1-2 раза в квартал, часто ищут подарочные издания, канцелярию |
| **Сценарии использования** | • Поиск по тематикам (подарки детям, коллегам)<br>• Покупка сопутствующих товаров |
| **Потребности** | Быстрый поиск по поводам, возрастным категориям |
| **Боли** | Непонятно, что подходит для конкретного случая |

### 3. Эпизодические/новые пользователи
| Характеристика | Описание |
|----------------|----------|
| **Демография** | Любой возраст |
| **Поведение** | Пришли по акции, за конкретной книгой из списка, по рекламе |
| **Сценарии использования** | • Единоразовая покупка<br>• Регистрация ради скидки 30% |
| **Потребности** | Простая навигация, выгодное первое предложение |
| **Боли** | Сложная регистрация, непонятные условия доставки |
