# Детальный анализ проблем пользователей Book24.ru

## Проблема 1: Плохо оптимизированная обработка пользовательских опечаток в поиске

### Описание проблемы
При вводе в поисковой строке названия книги с ошибкой пользователь видит сообщение:
> «К сожалению, по вашему запросу «ЗАПРОС» мы ничего не нашли.»

Хотя ниже в блоке «Вам может быть интересно» искомая книга часто присутствует.

### Наблюдаемые сценарии (на основе реального поведения сайта)
1. **Опечатка в названии книги**
   - *Пример:* Пользователь ввел «Марфий» вместо «Морфий» (книга М. Булгакова)
   - *Результат:* Сообщение «ничего не нашли», хотя книга есть в базе

2. **Опечатка в фамилии автора**
   - *Пример:* Пользователь ищет «Булгавков» вместо «Булгаков»
   - *Результат:* Поиск "съедает" фамилию и ищет только по имени «Михаил», выдавая книги всех авторов с именем Михаил

### Влияние на пользовательский опыт
- Пользователь тратит дополнительное время на исправление запроса
- Создается впечатление, что нужной книги нет на сайте
- Пользователь может уйти к конкуренту, даже если книга есть в наличии

### Корневые причины (на основе анализа)
- Отсутствует корректура текста (исправление опечаток) на уровне поискового движка
- Поиск по автору использует неточный алгоритм (частичное совпадение без приоритета точности)
- Блок «Вам может быть интересно» не интегрирован с основным поиском

## Проблема 2: Несогласованность контента в карточке товара

### Описание проблемы
На сайте отсутствует единый стандарт наполнения карточек книг. Это видно при сравнении разных книг.

| Тип контента | Пример: новинка/хит | Пример: нишевая книга |
|--------------|---------------------|------------------------|
| Ознакомительный фрагмент | ✅ Есть (например, у книги «Морфий») | ❌ Часто отсутствует |
| Отзывы | ✅ Есть несколько | ❌ Может не быть ни одного |
| Подробное описание | ✅ Развернутое | ❌ Только краткая аннотация |

*Примечание:* Это наблюдение основано на реальном просмотре разных книг на сайте.

### Влияние на пользовательский опыт
- Пользователь не может принять решение о покупке незнакомой книги
- Доверие вызывает только то, что уже популярно (хиты, бестселлеры)
- Нишевые, но качественные книги остаются незамеченными

### Корневые причины (гипотетические)
- Процесс наполнения карточек, вероятно, ручной и не имеет четких стандартов для всего ассортимента
- Приоритет отдается новинкам и потенциальным хитам
- Отсутствует автоматическое обогащение контента (например, парсинг аннотаций)
